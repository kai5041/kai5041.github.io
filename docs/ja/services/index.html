    How it works

    1. The Client contacts Operator Kai stating their needs.

    2. The Operator prepares and provides a minimum viable proposal for the Client.

    3. Once agreed, the Operator sends the Contract for the Client to sign (wet and digital signatures accepted).

    4. The Operator sends the Project Blueprint for the Client to evaluate. If approved, the Client must sign it.

    5. The commission is paid in installments defined by the Project Blueprint.

    6. The Operator provides proof of completion of each milestone.

    7. The Client pays the final remaining amount, and the Operator delivers the project.
